Для пользователей Windows
Подготовка данного проекта к запуску:
подготовка серверной части:
1)выгрузите векту main
2)перейдите в папку появившуюся папку Practice через cmd
3) введите: 
npm install
4) поставьте сервер очередей rabbitMq
5) всё, можно запускать, используя команду:
npm start

подготовка react-приложения:
1) выгрузите ветку react
2) установите глобально плагин create-react-app:
npm install create-react-app -g
3) перейдите в папку Practice
4) введите команду npm install
5) Запуск:
npm start

Запуск проекта:
1) npm start, находясь в папке серверной части (уровень package.json)
2) npm start, находяся в папке react-приложения(уровень package.json)
